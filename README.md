# 📘 Development & Reporting Knowledge Repository

This repository is created to store documentation, guides, and solutions related to our development environment, reporting tools, and portal systems.

The goal of this repository is to maintain a **central knowledge base** that can help both myself and junior developers quickly understand setups, troubleshoot issues, and recall solutions when needed.

---

## 🎯 Purpose of This Repository

* Store **important setup guides**
* Document **common issues and their solutions**
* Maintain **development and reporting knowledge**
* Help **junior developers learn the system faster**
* Keep a **reference for future troubleshooting**

---

## 📂 Contents of This Repository

The repository contains documentation related to:

* **SQL Server Database Management**
* **JasperSoft Studio Installation and Setup**
* **Workspace Configuration**
* **Reporting Issues and Fixes**
* **Development Issues**
* **Portal Site Issues and Solutions**
* **General Development Notes**

All detailed documentation is provided in the **DOC files uploaded in this repository**.

---

## 👨‍💻 How to Use This Repository

1. Browse the repository files.
2. Open the relevant **documentation file (.doc / .docx)**.
3. Follow the steps or solutions provided in the document.

These documents contain **step-by-step explanations, issue descriptions, and solutions** encountered during development and reporting work.
